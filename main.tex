\resumen{%
Este es un proyecto que trata el estudio de herramientas y técnicas usadas en algunos artículos sobre el reconocimiento de imágenes.

En el proyecto se ha estudiado con detenimiento las técnicas que se han usado en el reconocimiento de imágenes y, más específicamente, en el de la generación de frases a partir de una imagen. Se ha visto a través de este estudio que cualquier tarea que este relacionada con el reconocimiento de imágenes, es una tarea laboriosa y compleja. También se ha concluido con el estudio, la necesidad de modelos grandes y potentes para poder soportar la carga de computo de esta tarea, además de requerir un soporte físico con los suficientes recursos para soportar el modelo de manera efectiva y que permita obtener un rendimiento aceptable.

Este proyecto también cuenta con una aplicación de prueba, cuyo objetivo es mostrar los posibles usos de este tipo de herramientas y de su funcionamiento. Esta aplicación está construida con la idea de que sea usada por una persona invidente, por lo tanto su interfaz gráfica es dirigida a este tipo de usuario. Esto se ha realizado para dar una visión de las posibilidades que tiene el desarrollo de este tipo de herramientas y modelos.

En conclusión, se ha podido ver que este tipo de herramientas aún tienen un camino largo por desarrollar pero que han dado un paso más al conseguir evolucionar hacia la tarea de la generación de frases a partir de imágenes. Por lo tanto, estas herramientas se cree que irán evolucionando para abrir un nuevo abanico de posibilidades al desarrollo de aplicaciones para personas con ciertas discapacidades físicas.
}
\descriptores{}

\abstract{%
This project is about the study of tools and techniques used in some image recognition articles.

In this project we have studied in detail the techniques used on image recognition and more specifically in the image descriptions generation. We have seen with this study that any task related with image recognition is laborious and complex. Further we have concluded with this study that it's needed the existence of large models who can assume the processing of this kind of task. Also it's required a hardware with the enough resources who can assume the model in effective way and permit obtain an acceptable performance.

This project also count with a test application whose objective is show the possible uses of this kind of tools. Further it shows how this tools works. This application is built with the idea to be used by a blind person. Thus his user interface is aimed at this kind of user. This has been realized with the objective of give a view of the possibilities that the development of this kind of tools and models have.

In conclusion we have seen that this kind of tools still have a large way to go but it have gotten a new advance on image recognition because it now can produce sentence from an image. Thus we believe that this tools will evolve and at last will give us a new set of possibilities to develop applications and help disabled persons.
}
